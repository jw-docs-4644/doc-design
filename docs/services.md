# Services

Review your current documentation and create a roadmap for improvement. We assess:

- Documentation structure and organization
- Writing quality and consistency
- Tooling and automation gaps
- Team workflow and processes

Migrate your documentation to a sustainable, version-controlled workflow:

- Set up MkDocs, Sphinx, or other static site generators
- Establish writing standards and CI/CD pipelines
- Implement documentation-as-code in your development workflow
- Automate API documentation generation

## Technical Writing

Create clear, user-focused documentation:

- User guides and tutorials
- API documentation
- Architecture and design documentation
- Blog posts and thought leadership content

## Team Training

Get your team up to speed on docs-as-code practices:

- Workshop on tools and workflows
- Best practices for collaborative writing
- Git workflows for documentation
- Continuous improvement and feedback loops
